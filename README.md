# Arquitectura Landing (Next.js)

Sitio estático en Next.js para Alfredo Arvelo Arquitectura.

## Scripts

- `npm run dev` inicia entorno local
- `npm run build` genera build estático
- `npm run lint` valida código
- `npm run preview` sirve `out/` localmente

## Blog (MVP)

El blog usa Markdown como fuente en:

- `content/blog/*.md`

Hay una plantilla base en:

- `content/blog/_template.md`

Guía de operación semanal:

- `docs/blog-workflow.md`
